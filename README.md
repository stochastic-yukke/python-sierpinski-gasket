# シェルピンスキのギャスケットを表示するPythonプログラム
## 概要
[部誌を公開したリポジトリ](https://github.com/stochastic-yukke/sukenbushi2025) に関係した，シェルピンスキのギャスケットを表示するPythonプログラムを公開します。

mod2以外のシェルピンスキのギャスケットや，Kummerの定理を利用した3次元版のシェルピンスキのギャスケット，Lucasの定理の利用に関するベンチマークもあります。

Programs that generate different types of Sierpinski gaskets: -3D and modulo p gasket (for any prime p) using Kummer's Theorem; -Quickly generated modulo 2 gasket using Lucas's Theorem.
